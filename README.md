# 냉장고를 부탁해
## 📝 시작동기

늘어나는 음식물 쓰레기 배출량으로 인해 나라의 탄소 배출량이 상당하다고 들었습니다.
음식물 쓰레기의 배출을 줄이고자, 냉장고 관리 & 레시피 추천 애플리케이션을 만들기로 하였습니다.


## ✋ 역할

UI디자인, 안드로이드 애플리케이션 내 기능구현, 서버구축


## ⚒️ 사용기술

`java`, `Django`, `MySQL`, `retrofit`, `kakaoVisionAPI`
